# 🏦 Bank System (Java OOP)

A simple console-based Bank System built with Java to practice Object-Oriented Programming (OOP), Java Streams, and Clean Architecture principles.

## ✨ Features

* Create new bank accounts
* Find account by owner name
* Show all accounts
* Deposit money
* Withdraw money
* Transfer money between accounts
* View transaction history
* Operation result handling with success/error messages

## 📂 Project Structure
